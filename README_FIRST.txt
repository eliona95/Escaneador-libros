⚡ INICIO ULTRA-RÁPIDO:

1. Genera iconos:
   → Abre: generate-icons.html en tu navegador
   → Descarga: icon-192.png y icon-512.png

2. Sube a GitHub:
   → github.com → New repository → "escaner-libros"
   → Arrastra TODOS los archivos
   → Commit changes

3. Activa GitHub Pages:
   → Settings → Pages → Deploy from branch (main)
   → Guarda y espera 2 minutos

4. Instala en tu móvil:
   → Abre la URL en Safari (iPhone) o Chrome (Android)
   → Añadir a pantalla de inicio

5. ¡Usa la app!
   → Carga tu CSV
   → Escanea ISBNs
